  • What are the advantages and disadvantages of using external style sheets?

  Disadvantages of External Style Sheets

    1. Old Browsers - Some very old browsers (such Netscape Navigator 2) do not support style sheets.
    2. Browser dependency - formate is slightly difference on different browser.

  Advantages of External Style Sheets

    1. one change to the style sheet will change all linked pages.
    2. you can create classes of styles that can then be used on many different HTML elements.
