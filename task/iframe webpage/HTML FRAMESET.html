Iframe

- Iframe tag is use for embed video, audio or anykind of online Urls.

- Also use for webpage layout.

Frameset

frameset cols -

The cols attribute is used to create vertical frames in a web browser.
This attribute is basically used to define the no. of columns and their size inside the frameset tag.



frameset rows -

The rows attribute is used to create horizontal frames in the web browser.
This attribute is used to define the no. of rows and their size inside the frameset tag.
